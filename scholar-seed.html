Scholar Cache Seeder

Enter your current stats from your Google Scholar profile. Run this whenever your citation count changes.

⚠ First-time setup — fix your Firebase rules
In Firebase Console → Realtime Database → Rules, replace the rules with this and click Publish: 
{
  "rules": {
    "pulse": {
      "posts":    { ".read": true,  ".write": false },
      "meta":     { ".read": true,  ".write": false },
      "usedUrls": { ".read": false, ".write": false }
    },
    "scholar_cache": { ".read": true, ".write": true }
  }
}
After saving your stats below, change ".write": true back to ".write": false to lock it down again.

OR — paste this into Firebase Console directly

Go to Firebase Console → Realtime Database → Data, click the + button next to the root, add a node called scholar_cache and paste the JSON below (edit your numbers first): 

{
  "citations": 0,
  "hIndex": 0,
  "i10Index": 0,
  "source": "manual-seed",
  "fetchedAt": ""
}
